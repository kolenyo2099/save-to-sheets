2 Set up your Google Sheet

This extension saves results to a Google Sheet via a small script you deploy yourself — no login required, and you own all the data.

  1. Open (or create) the Google Sheet you want to save results into.
  2. Click Extensions → Apps Script in the menu bar.
  3. Delete any existing code and paste the script below, then click Save.
  4. Click Deploy → New deployment. Set type to Web app, execute as Me, access Anyone. Click Deploy.
  5. Authorize the script when prompted, then copy the Web app URL it gives you.
  6. Paste that URL into Step 3 below.
function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var lastCol = sheet.getLastColumn(); var headers = lastCol > 0 ? sheet.getRange(1, 1, 1, lastCol).getValues()[0] : []; return ContentService .createTextOutput(JSON.stringify({ headers: headers })) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { var lock = LockService.getScriptLock(); lock.waitLock(10000); var responseBody; try { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); if (data.action === 'getHeaders') { var lastCol = sheet.getLastColumn(); var headers = lastCol > 0 ? sheet.getRange(1, 1, 1, lastCol).getValues()[0] : []; responseBody = JSON.stringify({ status: 'ok', headers: headers }); } else if (data.action === 'setHeaders') { var h = data.headers || []; if (h.length > 0) { sheet.getRange(1, 1, 1, h.length).setValues([h]).setFontWeight('bold'); SpreadsheetApp.flush(); } responseBody = JSON.stringify({ status: 'ok' }); } else if (data.action === 'data' && Array.isArray(data.row)) { sheet.appendRow(data.row); SpreadsheetApp.flush(); responseBody = JSON.stringify({ status: 'ok' }); } else { sheet.appendRow([ data.id || '', new Date(), data.savedBy || '', data.query || '', data.title || '', data.url || '', data.snippet || '' ]); SpreadsheetApp.flush(); responseBody = JSON.stringify({ status: 'ok' }); } } finally { lock.releaseLock(); } return ContentService .createTextOutput(responseBody) .setMimeType(ContentService.MimeType.JSON); } function onOpen() { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); if (sheet.getLastRow() === 0) { sheet.appendRow(['ID', 'Timestamp', 'Saved By', 'Query', 'Title', 'URL', 'Snippet']); sheet.getRange(1, 1, 1, 7).setFontWeight('bold'); } }

⚙️ Advanced Features

Spreadsheet column structure

Define the columns your spreadsheet should have, and map each one to a data field the extension can capture. Click Apply to Sheet to write the header row, or Detect from Sheet to import existing headers from your sheet. Then click Save settings above to persist the mapping.

Available data fields

Fields marked Search only are only available when saving from Google Search results. Fields marked Save page are only available when using right-click → Save page to Sheets (they require reading the page source). Fields marked All contexts are always captured.

📤 Sharing with collaborators

To let others (students, colleagues) save to the same sheet, share the Apps Script URL and ask them to paste it into their extension settings along with their name. Each person's saves will appear in the sheet with their name in the Saved By column.

💡 The Apps Script is structure-agnostic — you only need to paste it once. Column layout is managed entirely in Advanced Features above and applied to your sheet via the Apply to Sheet button.
