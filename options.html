2 Set up your Google Sheet

This extension saves results to a Google Sheet via a small script you deploy yourself — no login required, and you own all the data.

  1. Open (or create) the Google Sheet you want to save results into.
  2. Click Extensions → Apps Script in the menu bar.
  3. Delete any existing code and paste the script below, then click Save.
  4. Click Deploy → New deployment. Set type to Web app, execute as Me, access Anyone. Click Deploy.
  5. Authorize the script when prompted, then copy the Web app URL it gives you.
  6. Paste that URL into Step 3 below.
function doPost(e) { const lock = LockService.getScriptLock(); lock.waitLock(10000); try { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const data = JSON.parse(e.postData.contents); sheet.appendRow([ data.id || "", new Date(), data.savedBy || "", data.query || "", data.title || "", data.url || "", data.snippet || "" ]); SpreadsheetApp.flush(); } finally { lock.releaseLock(); } return ContentService .createTextOutput(JSON.stringify({ status: "ok" })) .setMimeType(ContentService.MimeType.JSON); } function onOpen() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); if (sheet.getLastRow() === 0) { sheet.appendRow(["ID", "Timestamp", "Saved By", "Query", "Title", "URL", "Snippet"]); sheet.getRange(1, 1, 1, 7).setFontWeight("bold"); } }

3 Paste your Apps Script URL

This URL lets the extension write to your sheet. Keep it private — anyone with it can add rows.

📤 Sharing with collaborators

To let others (students, colleagues) save to the same sheet, share the Apps Script URL and ask them to paste it into their extension settings along with their name. Each person's saves will appear in the sheet with their name in the Saved By column.

💡 Tip: Add a header row to your sheet manually: ID · Timestamp · Saved By · Query · Title · URL · Snippet
